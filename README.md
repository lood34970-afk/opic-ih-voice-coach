# OPIc IH Voice Coach - Free Recording Mode

이 버전은 OpenAI API를 사용하지 않습니다. 별도 API 과금이 없습니다.

## 작동 방식

1. 문제 듣기
2. 답변 시작
3. 브라우저가 마이크로 답변을 녹음
4. 종료
5. 녹음 파일 다운로드
6. `ChatGPT 피드백 프롬프트 복사` 버튼 클릭
7. ChatGPT에 녹음 파일을 업로드하고 프롬프트를 붙여넣어 피드백 받기
